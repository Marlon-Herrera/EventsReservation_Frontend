# Events Reservations (Frontend)

Aplicación para gestionar eventos y reservas de los usuarios.

## 1. Configuración Inicial del Proyecto

### 1.1. Instalar Librerias

```sh
npm i
```

## 2. Iniciar Proyecto

### 2.1. Modo Desarrollo

```sh
npm run dev
```

### 2.2. Compilar proyecto

```sh
npm run build
```
